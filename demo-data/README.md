# Demo data

`legalpm-demo-data.json` is a **fictional** dataset for demoing the app — no real
client data. It's in the same `legalpm-data-export` format the app imports.

It populates every major area so you can show full functionality:
10 clients · 12 matters (Live / Pipeline / Closed / Lost) · budgets & line items ·
~10 months of financial snapshots per matter (so burn sparklines & the dashboard
trend render) · bills · WIP shaping proposal · 4 pricing proposals (versions,
items, AFAs) · 4 growth projects with tasks · quick tasks & slate · 20
distribution contacts with lists · 10 deal credentials · a report format.

## Load it

1. Open the app in **Chrome or Edge**.
2. (Recommended) Settings → Data Storage → **Clear All Data** first, for a clean slate.
3. Settings → Data Storage → **Import data from cloud export** → choose this file.
4. The app reloads with the demo data.

> Use a **separate browser profile** (or the hosted demo link) for demos so it
> never mixes with your real data. Everything stays local to that browser.

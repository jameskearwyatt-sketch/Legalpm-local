# Memory: features/pricing-proposal/category-breakdown-context
Updated: now

The 'Work Items' tab includes a 'Category Breakdown (Upper Estimate)' card that displays totals using `fee_upper` values. When multiple phases are defined, the breakdown shows per-phase category subtotals followed by an aggregate total. For single-phase or unphased proposals, only the aggregate breakdown is shown. Blue badges indicating active Alternative Fee Arrangements (AFAs) are displayed in the header to clarify context.

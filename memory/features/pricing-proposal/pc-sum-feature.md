# Memory: features/pricing-proposal/pc-sum-feature
Updated: now

Pricing proposal work items include a "PC Sum" (Provisional Contract Sum) checkbox feature. This flag indicates that the scope of work is unclear and the fee estimate is highly provisional—for example, when the number of entities to review is unknown. In the UI, a PC Sum tooltip explains the concept, and rows with PC Sum checked are displayed in violet/purple text. The Excel export includes a "PC Sum?" column immediately after the fee column with "Yes" for flagged items. If any PC Sum items exist in the proposal, an explanatory note is added to the footer explaining that these figures are indicative and subject to revision once the transaction structure is finalized.
